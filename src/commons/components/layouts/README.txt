Los Layouts son componentes que solo aportan un template con uno o multiples slots,
su mision es evitar que las vistas se preocupen del posicioneamiento de sus elementos.

Cada vista utilizará al menos un layout pudiendo combinar varios si lo necesitase.

Esto facilitará la refactorizaciónes visuales de las vistas al quedar catalogadas de facto
por layouts se espera el mismo tipo de problemas para cada uno de ellos siendo resuletos de forma centralizada

Nomenclatura de layout: layout-nombredescriptivo.vue
